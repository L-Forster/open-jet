# Installation

## Python package

```bash
pip install open-jet
```

## Runtime requirements

- `llama-server` from `llama.cpp` (default runtime)
- a local `.gguf` model file, or `ollama` installed for model download

For build instructions by environment, see deployment docs:

- [Jetson setup](deployment/jetson.md)
- [Linux x86 + NVIDIA](deployment/linux-x86-nvidia.md)
- [CPU-only](deployment/cpu-only.md)
