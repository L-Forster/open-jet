# Configuration

Main settings are stored in `config.yaml`, including:

- context window size
- memory guard limits
- session manifest retention settings
- telemetry collector/export settings
- session state/resume settings

Runtime-specific configuration examples are documented under:

- [llama.cpp runtime](runtimes/llama-cpp.md)
- [SGLang runtime](runtimes/sglang.md)
- [TensorRT-LLM runtime](runtimes/tensorrt-llm.md)

Telemetry-specific behavior is documented in:

- [Session state and logging](usage/session-state-and-logging.md)
- [Telemetry](telemetry.md)
